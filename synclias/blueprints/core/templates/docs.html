Technitium Configuration Guide

Overview

Synclias needs very minimal access to Technitium, with permissions only to eject a site from the cache.

This guide will:

  • Create a User named "cache_clear"
  • Create a Group called "cache_clear_group"
  • Add the user to the group
  • Grant necessary permissions to the group 
  • Create a user token for Synclias to use

All steps below assume access to the management web interface

User Setup

1. Navigate to: Administration -> Users

2. Click Add User and create:

  • Display Name: cache_clear
  • Username: cache_clear
  • Password: <Supply a very long password, this will never be needed>

3. Click Add

Group Setup

1. Navigate to: Administration -> Groups

2. Click Add Group and create:

  • Name: cache_clear_group
  • Description: "Group for Synclias cache management users

3. Click Add

Add User to Group

1. Navigate to: Administration -> Groups

2. Click the "cache_clear_group"

3. Use the Add User dropdown to add cache_clear to the group

4. Click Save

Grant Required Permissions

1. Navigate to: Administration -> Permissions

2. In the Section column, click Cache

3. From the Add Group dropdown, select the cache_clear_group to add it to the Group Permissions section

4. Ensure permissions for the cache_clear_group are set as follows:

  • View: Empty
  • Modify: Empty
  • Delete: Checked

5. Click Save

Create a Token for the Account

1. Navigate to: Administration -> Sessions

2. Click Create Token and create:

  • Username: select cache_clear from the dropdown
  • Token Name: "Synclias Cache Clear"

4. Click Create

5. Note down the Token String. Once this window is closed there is no way to get the token displayed again

6. Click Close

Note: If the token is lost at any time or needs to be replaced, delete the token from the Sessions page and create another
